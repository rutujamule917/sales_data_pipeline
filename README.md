# Sales Data Pipeline Project

## Objective
Build a data pipeline to process sales data and generate business insights.

## Tools Used
- Python
- PySpark
- CSV

## Steps Performed
1. Data Ingestion from CSV
2. Data Cleaning (null & duplicates removed)
3. Data Transformation (created total_amount column)
4. Aggregation:
   - Total Sales
   - Sales by Category
   - Sales by Region
   - Top Products

## Business Insights
- Electronics category generated highest revenue
- West region performed best
- Mobile is top-selling product

## Output
Processed data saved in CSV format.

## How to Run
1. Install PySpark
2. Run: python pipeline.py
